# 中国区域单独展示实现方案

本文档记录了地图告警功能中**中国区域单独展示**的完整实现方式，
供后续需要恢复时参考。

---

## 一、功能概述

在"地图告警"浮层中，上半部分单独展示中国地图（含省界+南海插图），
下半部分展示世界地图。两张图各自渲染对应范围内的机场告警散点。

---

## 二、涉及文件

| 文件 | 路径 | 说明 |
|------|------|------|
| `map.js` | `mtws_django/static/js/map.js` | 地图功能核心 JS |
| `index.html` | `mtws_django/templates/web/index.html` | 浮层 DOM 结构 |
| `main.css` | `mtws_django/static/css/main.css` | 浮层布局样式 |
| `china.json` | `mtws_django/static/geo/china.json` | 中国省级边界 GeoJSON（35 个 feature） |
| `china_sea.json` | `mtws_django/static/geo/china_sea.json` | 南海诸岛/九段线专用 GeoJSON |

---

## 三、GeoJSON 数据说明

### `china.json`
- 格式：GeoJSON `FeatureCollection`，共 **35 个 feature**（34 省级行政区 + 南海相关）
- 来源风格：阿里云 DataV GeoAtlas / 国标 adcode 省级数据
- 关键字段：`adcode`（如 `110000`=北京）、`name`（如 `北京市`）、`level: "province"`
- 南海九段线：`adcode: "100000_JD"`，`name: ""`（空字符串），在主图中需隐藏
- 坐标系：**标准 WGS84**，无需经度偏移

### `china_sea.json`
- 专门用于南海插图，只含南海诸岛/九段线范围的 feature
- `adcode: "100000_JD"`

## 六、map.js 完整实现

### 6.1 变量声明

在文件顶部，需要以下变量：

```javascript
let mapAlertEnabled  = false;
let _mapCoordCache   = null;
let _chinaChart      = null;   // 中国图表实例
let _worldChart      = null;
let _chinaGeoReady   = false;  // china.json 已注册
let _chinaSeaGeoReady = false; // china_sea.json 已注册
let _worldGeoReady   = false;
```

### 6.2 GeoJSON 加载（`_initMapCharts`）

同时加载三个 GeoJSON 文件：

```javascript
function _initMapCharts() {
    const base = (window.staticUrl || '/static/') + 'geo/';
    const jobs = [];

    if (!_chinaGeoReady) {
        jobs.push(
            fetch(base + 'china.json')
                .then(r => { if (!r.ok) throw new Error('china.json 加载失败'); return r.json(); })
                .then(data => { echarts.registerMap('mtws_china', data); _chinaGeoReady = true; })
        );
    }

    if (!_chinaSeaGeoReady) {
        jobs.push(
            fetch(base + 'china_sea.json')
                .then(r => { if (!r.ok) throw new Error('china_sea.json 加载失败'); return r.json(); })
                .then(data => { echarts.registerMap('mtws_china_sea', data); _chinaSeaGeoReady = true; })
        );
    }

    if (!_worldGeoReady) {
        jobs.push(
            fetch(base + 'world_pacific.json')
                .then(r => { if (!r.ok) throw new Error('world_pacific.json 加载失败'); return r.json(); })
                .then(data => { echarts.registerMap('mtws_world_pacific', data); _worldGeoReady = true; })
        );
    }

    Promise.all(jobs)
        .then(() => { _createInstances(); _fetchCoordsAndRender(); })
        .catch(err => console.error('[地图告警] GeoJSON 加载失败:', err));
}
```

### 6.3 实例创建（`_createInstances`）

```javascript
function _createInstances() {
    _syncPanelPosition();

    const chinaEl = document.getElementById('map-china');
    const worldEl = document.getElementById('map-world');

    if (chinaEl && !_chinaChart) {
        _chinaChart = echarts.init(chinaEl, null, { renderer: 'canvas' });
        _chinaChart.setOption(_buildChinaOption(chinaEl.offsetWidth, chinaEl.offsetHeight));
    }

    if (worldEl && !_worldChart) {
        _worldChart = echarts.init(worldEl, null, { renderer: 'canvas' });
        _worldChart.setOption(_buildWorldOption());
    }

    setTimeout(() => {
        if (_chinaChart) _chinaChart.resize();
        if (_worldChart) _worldChart.resize();
    }, 200);
}
```

### 6.4 面板定位（`_syncPanelPosition`）

上下各半屏，需同时为中国图和世界图分配高度：

```javascript
function _syncPanelPosition() {
    const panel = document.getElementById('map-alert-panel');
    if (!panel) return;

    const titleRow = document.querySelector('.content-section > .title-row');
    const topPx = (titleRow && titleRow.getBoundingClientRect().bottom > 0)
        ? Math.round(titleRow.getBoundingClientRect().bottom)
        : 242;

    const totalH = window.innerHeight - topPx;
    const halfH  = Math.floor(totalH / 2);

    panel.style.top    = topPx + 'px';
    panel.style.height = totalH + 'px';

    const chinaEl = document.getElementById('map-china');
    const worldEl = document.getElementById('map-world');
    if (chinaEl) chinaEl.style.height = halfH + 'px';
    if (worldEl) worldEl.style.height = (totalH - halfH) + 'px';

    if (_chinaChart) _chinaChart.resize();
    if (_worldChart) _worldChart.resize();
}
```

### 6.5 中国地图 ECharts Option（`_buildChinaOption`）

核心：双 `geo` 层（主图 + 南海插图）+ `graphic` 绘制小图边框。

```javascript
function _buildChinaOption(containerW, containerH) {
    const W = containerW || 680;
    const H = containerH || 300;
    const minDim = Math.min(W, H);

    // 南海插图尺寸计算
    const lngSpan    = 18.0;
    const latSpan    = 21.3;
    const aspect     = lngSpan / latSpan;   // ≈ 0.845（宽/高）
    const insetSizePct = 0.30;
    const insetH     = Math.round(minDim * insetSizePct);
    const insetW     = Math.round(insetH * aspect);
    const cx         = Math.round(W * 0.88);
    const cy         = Math.round(H * 0.82);
    const margin     = 5;

    const boxLeft   = cx - Math.round(insetW / 2) - margin;
    const boxTop    = cy - Math.round(insetH / 2) - margin;
    const boxWidth  = insetW + 2 * margin;
    const boxHeight = insetH + 2 * margin;

    return {
        backgroundColor: '#1b2838',
        title: {
            text: '中国',
            left: 'center', top: 2,
            textStyle: { color: '#8fa8c0', fontSize: 11, fontWeight: 'normal' }
        },
        tooltip: _makeTooltip(),
        geo: [
            {
                // 主图：裁剪至内地范围（boundingCoords 排除南海小图区域）
                map: 'mtws_china',
                roam: true,
                zoom: 1.1,
                silent: true,
                label: { show: false },
                emphasis: { disabled: true },
                boundingCoords: [[73.5, 53.5], [135.1, 17.8]],
                itemStyle: {
                    areaColor: '#1e3348',
                    borderColor: '#3d6680',
                    borderWidth: 0.6
                },
                // 隐藏南海诸岛/九段线（name='' adcode=100000_JD）
                regions: [{
                    name: '',
                    itemStyle: { opacity: 0, borderWidth: 0 },
                    label: { show: false }
                }]
            },
            {
                // 南海插图：右下角，专用 GeoJSON
                map: 'mtws_china_sea',
                layoutCenter: ['88%', '82%'],
                layoutSize: (Math.round(insetSizePct * 100)) + '%',
                roam: false,
                silent: true,
                label: { show: false },
                emphasis: { disabled: true },
                boundingCoords: [[104.5, 3.2], [122.5, 24.5]],
                itemStyle: {
                    areaColor: '#1e3348',
                    borderColor: '#3d6680',
                    borderWidth: 0.5
                }
            }
        ],
        graphic: [
            {
                id: 'inset-border',
                type: 'rect', z: 100,
                left: boxLeft, top: boxTop,
                shape: { width: boxWidth, height: boxHeight },
                style: {
                    fill: 'rgba(22, 35, 50, 0.75)',
                    stroke: '#4a7c99',
                    lineWidth: 1.5
                }
            },
            {
                id: 'inset-label',
                type: 'text', z: 101,
                left: boxLeft + 2,
                top: boxTop - 15,
                style: {
                    text: '南海诸岛',
                    fill: '#7a9ab8',
                    font: '10px sans-serif'
                }
            }
        ],
        series: [{
            name: 'airports',
            type: 'scatter',
            coordinateSystem: 'geo',
            geoIndex: 0,
            symbolSize: 7,
            data: [],
            label: { show: false },
            emphasis: { disabled: true }
        }]
    };
}
```

### 6.6 主更新入口（`updateMapAlert`）

同时更新中国图和世界图的散点数据：

```javascript
function updateMapAlert() {
    if (!mapAlertEnabled) return;

    if (!_chinaChart || !_worldChart) {
        if (_chinaGeoReady && _chinaSeaGeoReady && _worldGeoReady) {
            _createInstances();
        } else {
            return;
        }
    }

    if (_mapCoordCache === null) {
        _fetchCoordsAndRender();
        return;
    }

    // false = 使用原始 WGS84 经度（中国图）
    // true  = 转换为 pacific 经度（世界图）
    _chinaChart.setOption({ series: [{ data: _buildMarkers(false) }] });
    _worldChart.setOption({ series: [{ data: _buildMarkers(true)  }] });
}
```

### 6.7 销毁（`_destroyMapCharts`）

```javascript
function _destroyMapCharts() {
    if (_chinaChart) { _chinaChart.dispose(); _chinaChart = null; }
    if (_worldChart) { _worldChart.dispose(); _worldChart = null; }
}
```

### 6.8 Resize 处理

窗口 resize 时需同时处理中国图的 `graphic` 重新计算：

```javascript
let _resizeTimer = null;
window.addEventListener('resize', () => {
    if (_resizeTimer) clearTimeout(_resizeTimer);
    _resizeTimer = setTimeout(() => {
        _syncPanelPosition();
        if (_chinaChart) {
            const chinaEl = document.getElementById('map-china');
            if (chinaEl) {
                _chinaChart.setOption({
                    graphic: _buildChinaOption(chinaEl.offsetWidth, chinaEl.offsetHeight).graphic
                });
            }
            _chinaChart.resize();
        }
        if (_worldChart) _worldChart.resize();
    }, 100);
});
```

## 七、`_buildMarkers` 函数说明

```javascript
/**
 * @param {boolean} transformWorld  是否将经度转换为 world_pacific 坐标系
 */
function _buildMarkers(transformWorld) {
    if (!_mapCoordCache || !filteredAirportData) return [];
    const markers = [];
    filteredAirportData.forEach(airport => {
        const code   = airport.airport_4code;
        const coords = _mapCoordCache[code];
        if (!coords) return;
        const level = getHighestAlertLevel(airport);
        const color = getAlertColor(level);
        const lon   = transformWorld ? _toLonPacific(coords.lon) : coords.lon;
        markers.push({
            name: code,
            value: [lon, coords.lat],
            itemStyle: { color: color, opacity: 0.9 }
        });
    });
    return markers;
}
```

- 中国图调用：`_buildMarkers(false)`（WGS84 原始经度，直接对应省界坐标）
- 世界图调用：`_buildMarkers(true)`（经度减 150°，对应 world_pacific.json 坐标系）

## 九、关键技术要点

- **双 geo 层**：ECharts 支持 `geo` 为数组，`series.geoIndex: 0` 绑定散点到主图
- **南海独立 GeoJSON**：若直接用 `china.json` 的九段线渲染插图，其他省份会泄漏到插图范围外；使用独立的 `china_sea.json` 可避免此问题
- **中国图坐标不需偏移**：`china.json` 使用标准 WGS84，散点传入原始经纬度即可（`_buildMarkers(false)`）
- **南海主图隐藏**：通过 `regions: [{ name: '', itemStyle: { opacity: 0 } }]` 将 `adcode=100000_JD`（name 为空字符串）的 feature 设为不可见
